# obs_interpret_udunits_time

return list of time vars

## Usage

``` r
obs_interpret_udunits_time(vals, unitstring, tz = "UTC")
```

## Arguments

- vals:

  time array

- unitstring:

  month

- tz:

  time zone, "UTC" default

## Value

return list of time vars
