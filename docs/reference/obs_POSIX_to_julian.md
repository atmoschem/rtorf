# obs_POSIX_to_julian

return time

## Usage

``` r
obs_POSIX_to_julian(times, epoch = ISOdatetime(1, 1, 1, 0, 0, 0, tz = "UTC"))
```

## Arguments

- times:

  time

- epoch:

  default 0001-01-01

## Value

return dates

## Note

Convention is that the Julian date of the epoch is 1.0 (not 0!)
