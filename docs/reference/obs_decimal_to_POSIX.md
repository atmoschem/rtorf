# obs_decimal_to_POSIX

return list of time vars

## Usage

``` r
obs_decimal_to_POSIX(decimal.dates, tz = "UTC", legacy = TRUE)
```

## Arguments

- decimal.dates:

  vecot with format YYYY.yyyy

- tz:

  time zone, "UTC" default

- legacy:

  logical, to use legacy code

## Value

return list of time vars
